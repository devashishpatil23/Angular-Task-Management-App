Task Management Application - Features

  • 🚀 Built a Task Management CRUD Application using Angular 19 and Firebase.
  • 🔐 Implemented Firebase Authentication for secure sign-up and login.
  • 🔄 Used Auto Login on page refresh and Auto Logout on session expiry.
  • 💾 Integrated Firestore / Realtime Database for task management with full CRUD functionality.
  • ⚡ Applied Standalone Components for a modular and optimized architecture.
  • ⚡ Implemented Lazy Loading to improve app performance.
  • 🔀 Used Child Routes for better navigation structure.
  • 🛡️ Secured routes with Route Guards, using CanActivate to prevent unauthorized access.
  • 🔔 Used ngx-toastr for displaying success, error, and notification messages.
  • 🌍 Implemented HTTP Interceptors for handling authentication and error responses globally.
  • ⚡ Used Angular Signals for efficient state management and reactive UI updates.
